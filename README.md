# SEO Assistant

SEO Assistant is a full-stack web application that helps users improve their website's SEO through keyword research, meta tag generation, and website audits.

## Features

* User Authentication
* SEO Website Audits
* Keyword Research
* Meta Tag Generation
* Secure Dashboard Access

## Tech Stack

**Frontend:** Next.js, React, Tailwind CSS
**Backend:** Node.js, Express.js, MongoDB, JWT

## Installation

```bash
# Backend
cd server
npm install
npm run dev

# Frontend
cd client
npm install
npm run dev
```

## Deployment

* Frontend: Vercel
* Backend: Render
* Database: MongoDB Atlas
